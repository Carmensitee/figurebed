# 工作日志 — 2026-06-11

## 修复：一键导入全部历史数据 → 回测无数据

**根因**：东方财富 `push2his` K 线 API 对这个网络环境统一拒连（`RemoteDisconnected`），所有指数历史日线导入失败。没了指数数据，回测中每个净值日期都因 `t_date not in index_map` 被跳过，结果为空。

**现象**：
- 161124（港股小盘LOF）：HSSI 指数拉不到
- 501227（红利优选LOF）：000922 中证红利也拉不到
- 基金净值 API（`api.fund.eastmoney.com`）正常，场内价格（Sina K 线）正常

000922 完整覆盖：2005-01-04 ~ 2026-06-11，共 5205 条。

### 错误上报修复

`auto_import_fund_nav()`：API 异常原来被 `except: break` 吞掉，用户看到「无新数据」误以为成功。现在返回 HTTP 状态码、超时、API ErrCode 等具体错误信息。

`auto_import_fund_prices()`：Sina API 返回非预期格式时同样上报错误。

### 数据管理页面 UI 修复

导入结果改用 `st.session_state` 持久化，解决 `st.rerun()` 后结果一闪而过的问题。
