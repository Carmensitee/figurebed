# 基金买入赎回跟踪与收益计算工具 v1.0

## 是什么

一个纯网页工具，用来跟踪单只基金的买入批次、模拟 FIFO 赎回、计算实际收益。

7 天免惩罚赎回日的计算逻辑，来自 `fund-redemption-calc` 项目。

## 怎么用

1. 双击 `index.html`，浏览器打开
2. 顶部下拉框选择基金（先选一只）
3. 切换到「买入管理」→ 添加几笔买入记录
4. 切换到「持仓汇总」→ 看总剩余和免罚份额
5. 切换到「赎回模拟」→ 输入日期和份额，看 FIFO 匹配结果
6. 基金净值公布后，切换到「实际赎回」→ 录入数据，自动算收益
7. 切换到「赎回记录」→ 看历史赎回和累计收益

## 数据存在哪里

浏览器 `localStorage`，关掉页面数据不会丢。换浏览器或清缓存会丢失。

## 核心规则

| 规则 | 说明 |
|------|------|
| FIFO 先进先出 | 赎回时从最早的买入批次开始扣减 |
| 7 天惩罚线 | 买入确认后持有 >= 7 自然日，赎回不收 1.5% 惩罚费 |
| 场内买入 | 成交即确认，持有从买入日起算 |
| 场内申购 | 等申购确认日，持有从确认日起算 |
| 惩罚费估算 | 被扣份额 × 单位成本 × 1.5% |

## 收益计算公式

```
对应买入成本 = 被赎回份额 ÷ 买入份额 × 买入金额（含手续费，按比例分摊）
实际收益 = 实际到账金额 - 对应买入成本
实际收益率 = 实际收益 ÷ 对应买入成本
```

## 节假日数据

节假日列表硬编码在 `index.html` 中（`HOLIDAYS` 变量），覆盖 2026 年 A 股/港股/美股。如需更新，编辑 `index.html` 中的节假日数据即可。
